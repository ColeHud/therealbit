Welcome to BitCatcher, a thrilling game that requires some luck and chance.

BitCatcher revolves around a character that scurries left and right to catch money. When money is wasted and isn't caught, the character loses a life. Eventually it's game over.

Despite being a game, BitCatcher requires a credit to play. Each credit is worth .0010 bitcoins aka 25 cents. At the end of the day, the winner with the highest score takes the entire pot.

This game is made to allow users to learn about the world of Bitcoin. Users are required to create a Blockchain Wallet and learn the value of bitcoin and how conserving money is an important trait that should eliminte wasteful spending habits.
